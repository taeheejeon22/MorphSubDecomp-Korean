# tokenization stratagies
## *baselines*
### eojeol
- composed                WP only
- decomposed pure (nfd)   KR-BERT

### morpheme_orig
- composed                kortok
- decomposed pure

### morpheme_fixed
- (decomposed pure)


## *OUR*
### morpheme_orig
- (composed + grammar_symbol)
- (grammar_symbol + decomposed_lexical)

- (grammar_symbol + decomposed pure)
    - (composed + grammar_symbol + decomposed_grammatical)

### morpheme_fixed
- composed
- composed + grammar_symbol
- grammar_symbol + decomposed_lexical

- (grammar_symbol + decomposed pure)
    - (composed + grammar_symbol + decomposed_grammatical)
